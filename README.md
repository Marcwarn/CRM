# Doings Sales Intelligence

AI-drivet CRM-system för Doings konsulter.

## Features
- Pipeline-översikt per stage
- AI-driven input (skriv eller tala in deals)
- CSV-import från Slack säljpipe
- Automatiska insights och varningar
- Team, konsult och ledningsvy

## Användning
Öppna index.html i webbläsare eller gå till GitHub Pages URL.
